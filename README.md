# 🍅 极简番茄钟 (Minimalist Pomodoro)

一款追求极致美学、护眼与防卡死的 Chrome/Edge 浏览器番茄钟插件。

## ✨ 特色
*   **液态流体动效**：通过水位上升直观感知时间积累。
*   **哑光科技风 UI**：精心调制的珊瑚红与蒂芙尼蓝配色，支持深色/护眼亮色双主题。
*   **防卡死设计**：通知 10 秒后自动执行下一步（如自动开始休息），无需手动点击。
*   **极致轻量**：体积 < 20KB，零内存负担。

## 🔧 使用说明
*   点击右上角 ☀ 切换日间/夜间模式。
*   点击时间下方的数字直接修改工作/休息时长。
*   好好享受专注时光！
